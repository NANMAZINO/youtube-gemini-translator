# 🤖 YouTube AI Translator 사용자 가이드

> **"언어의 장벽 없이, 문맥을 이해하는 AI와 함께 YouTube를 즐기세요."**

**YouTube AI Translator**는 단순한 번역기가 아닙니다. Google의 최신 **Gemini 3 Flash** AI 모델을 활용하여, 영상의 전체 문맥을 이해하고 자연스러운 한국어 자막을 실시간으로 제공하는 Chrome 확장 프로그램입니다.

---

## 🌟 소개 (Introduction)

### 왜 YouTube AI Translator인가요?

기존의 자동 번역은 문장 단위로 끊어서 번역하기 때문에 문맥이 끊기거나, 대명사가 무엇을 가리키는지 알 수 없는 경우가 많았습니다. **YouTube AI Translator**는 다릅니다.

- **🧠 문맥 인식 (Context-Aware):** 앞뒤 내용을 기억하여 자연스럽게 이어지는 번역을 제공합니다.
- **⚡ 실시간 스트리밍:** AI가 번역하는 즉시 영상 위에 오버레이 자막으로 보여줍니다.
- **🎯 정밀 재분할 (Refine):** 뚝뚝 끊기는 자동 생성 자막을 문장 단위로 깔끔하게 모아서 다시 배치합니다.
- **💰 합리적인 비용:** Google Gemini API의 무료 티어(Free Tier)를 활용하여 개인 사용자는 대부분 무료로 이용 가능합니다.

---

## 🚀 설치 방법 (Installation)

현재 이 프로그램은 개발자 모드를 통해 설치할 수 있습니다.

1. **소스 코드 준비**: 전달받은 압축 파일(ZIP)을 풀거나 Git으로 코드를 다운로드합니다.
2. **Chrome 확장 프로그램 관리 페이지 접속**:
   - 주소창에 `chrome://extensions` 입력
   - 또는 우측 상단 메뉴(⋮) > 확장 프로그램 > 확장 프로그램 관리
3. **개발자 모드 켜기**: 우측 상단 **[개발자 모드]** 토글 스위치를 **ON**으로 켭니다.
4. **확장 프로그램 로드**:
   - 좌측 상단 **[압축해제된 확장 프로그램을 로드합니다]** 버튼 클릭
   - 압축을 푼 폴더 안의 **`extension`** 폴더를 선택합니다.
5. **완료**: 목록에 **"YouTube AI Translator"** 카드가 나타나면 설치 성공!
   - _팁: 상단 퍼즐 아이콘(🧩)을 눌러 핀(📌)을 고정해두면 편리합니다._

## 🔑 시작하기 (Getting Started)

사용을 위해서는 Google의 Gemini API Key가 필요합니다. (1분 소요)

### 1단계: API Key 발급

1. [Google AI Studio](https://aistudio.google.com/apikey)에 접속하여 Google 계정으로 로그인합니다.
2. **"Create API Key"** 버튼을 클릭합니다.
3. 생성된 키(문자열)를 복사합니다.

### 2단계: 확장 프로그램에 키 등록

1. Chrome 우측 상단 툴바에서 **YouTube AI Translator 아이콘**을 클릭합니다.
2. 팝업 창의 **"Gemini API Key"** 입력란에 복사한 키를 붙여넣습니다.
3. **[저장]** 버튼을 클릭합니다.

> **보안 안심:** 등록된 API Key는 사용자 브라우저 내부에만 난독화되어 저장되며, 외부 서버로 절대 전송되지 않습니다.

## 📖 사용 방법 (User Manual)

### 1. 자막 패널 열기

YouTube 영상 페이지에 들어가면, 영상 하단 좋아요/공유 버튼 옆에 **"📜 스크립트 열기"** 버튼이 생깁니다. 이 버튼을 누르면 우측에 자막(스크립트) 패널이 자동으로 열립니다.

### 2. AI 번역 시작

자막 패널 상단에 추가된 **"🤖 AI 번역"** 버튼을 클릭하세요.

- AI가 자막을 읽고 번역을 시작합니다 (실시간 진행률 표시).
- 번역된 내용은 **영상 화면 위에 보기 좋게 오버레이**됩니다.
- 기존 자막 패널에서도 번역된 텍스트를 확인할 수 있습니다.

> [!TIP]
> **오버레이 자막 커스텀:**
>
> - **위치 변경:** 자막 오버레이를 마우스로 드래그하여 원하는 위치로 옮길 수 있습니다.
> - **크기 조절:** 오버레이 위에 마우스를 올리고 `Ctrl + 휠`을 돌리면 자막 크기가 실시간으로 조절됩니다.

### 3. 자막 재분할 (Refine)

자동 생성 자막은 단어가 뚝뚝 끊겨 있어 읽기 불편할 때가 많습니다. 번역이 완료되면 AI 번역 버튼 옆에 **"자막 재분할"** 버튼이 활성화됩니다.

- 버튼을 누르면 AI가 문맥에 맞춰 자막의 타임라인을 다시 조정합니다.
- 한 눈에 의도를 파악할 수 있는 깔끔한 문장으로 정리됩니다.

### 4. 설정 조절 (옵션)

확장 프로그램 아이콘을 눌러 팝업 메뉴에서 설정을 변경할 수 있습니다.

- **추론 레벨 (Thinking Level):**
  - **Minimal (기본):** 빠르고 경제적입니다. 대부분의 영상에 적합합니다.
  - **Low/Medium/High:** 복잡한 내용이나 여러 화자가 섞인 영상에서 더 똑똑하게 번역합니다. (시간과 비용이 조금 더 들 수 있습니다.)
- **언어 설정:** 원본 언어(자동 감지)와 번역될 언어(한국어, 영어, 일본어)를 설정할 수 있습니다.

### 5. 데이터 내보내기 및 가져오기 (고급)

번역된 데이터를 파일로 저장하거나, 다른 기기에서 가져와서 사용할 수 있습니다.

- **내보내기:** 자막 패널의 **[내보내기]** 버튼을 누르면 현재 번역된 내용을 JSON 파일로 저장합니다.
- **가져오기:** 영상 하단에 주입된 **[가져오기]** 버튼을 통해 저장해둔 JSON 파일을 불러와서 즉시 적용할 수 있습니다.

### 6. 태스크 관리

- **자동 중지:** 번역 도중 다른 탭으로 이동하거나 다른 영상을 클릭하면, 배터리와 리소스 절약을 위해 진행 중인 번역을 자동으로 즉시 중단합니다.

## 💡 자주 묻는 질문 (FAQ)

**Q. 자막이 없는 영상도 번역되나요?**
A. 아니요. YouTube 자체 자막(자동 생성 자막 포함) 데이터가 있어야 번역이 가능합니다. 자막 데이터가 아예 없는 영상은 지원하지 않습니다.

**Q. 비용은 얼마나 드나요?**
A. Google Gemini API는 개인 사용자에게 **상당한 양의 무료 사용량(Free Tier)**을 제공합니다. 일반적인 YouTube 시청 패턴으로는 무료 한도 내에서 충분히 사용 가능합니다.

- 팝업 메뉴의 **"토큰 사용량"** 탭에서 실시간 예상 비용(무료 티어 초과 시 기준)을 확인할 수 있습니다.

**Q. 번역이 멈췄어요.**
A. 일시적인 네트워크 오류나 API 제한일 수 있습니다.

- "AI 번역" 버튼이 "재시도" 상태로 바뀌면 다시 클릭해주세요.
- 자동으로 최대 3회까지 재시도를 수행합니다.

**Q. 영상을 다시 볼 때마다 번역해야 하나요?**
A. 아니요! 한 번 번역된 내용은 **30일간 내 컴퓨터에 자동 저장(캐시)**됩니다. 다시 방문하면 즉시 번역된 자막이 뜹니다.

## 🛠 문제 해결

- **403 에러 (Permission Denied):** API Key가 올바르지 않거나 만료되었습니다. 키를 다시 발급받아 등록해주세요.
- **429 에러 (Resource Exhausted):** 단시간에 너무 많은 요청을 보냈습니다. 잠시(1~2분) 기다렸다가 다시 시도해주세요.
